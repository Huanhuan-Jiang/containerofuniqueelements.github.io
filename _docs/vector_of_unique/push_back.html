bool push_back(const T& value);
(1)
bool push_back( T&& value);
(2)

The push_back method adds a value to the back of the vector if it is not already present. This ensures that all entries in the vector remain unique.

1) The new element is initialized as a copy of value.
2) value is moved into the new element.

All iterators (including the cend() iterator) are invalidated. No references are invalidated.

Parameters

value - The value of the element to append.
Type Requirements
- T must meet the requirements of CopyAssignable in order to use overload (1).
- T must meet the requirements of MoveAssignable in order to use overload (2).

Return Value

Returns true if the appendage is successful, and false otherwise.

Complexity

Constant.

Exceptions

If an exception is thrown for any reason, this function has no effect (strong exception safety guarantee).
