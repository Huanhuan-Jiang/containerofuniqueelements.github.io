void swap( vector_of_unique& other );

Exchanges the contents of the container with those of other. Does not invoke any move, copy, or swap operations on individual elements.

All iterators and references remain valid. The cend() iterator is invalidated.

Parameters

other - container to exchange the contents with

Return value

(none)

Complexity

Constant

Exceptions

Any exception thrown by the swap of the Hash or KeyEqual objects.

(until C++17)
2) If the allocator or key type does not support noexcept, the exceptions thrown by std::vector::swap or std::unordered_set::swap may propagate. However, these cases are rare when using the default allocator and the move constructor of T is noexcept.
