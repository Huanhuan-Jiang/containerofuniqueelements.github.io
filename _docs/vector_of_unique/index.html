Defined in header <vector_of_unique>
template <

   class  T,
   class  Hash  = std::hash
   class  KeyEqual  = std::equal_to
> class vector_of_unique ;

Combining the characteristics of a double-ended queue (std::vector) and an std::unordered_set, the vector_of_unique ensures that all elements are unique and can be accessed or modified from both ends. This makes it perfect for applications where maintaining order and preventing duplicates is essential.

      // Create a vector_of_unique containing integers
      containerofunique::vector_of_unique<int> unv = {3, 5, 20, 9};
       
      // Add an integer to the beginning and end of the vector_of_unique
      unv.push_back(28);
       
      // print values of vector_of_unique
