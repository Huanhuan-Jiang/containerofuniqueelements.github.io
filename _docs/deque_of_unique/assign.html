template< class InputIt >
void assign( InputIt first, InputIt last );
(1)
void assign( std::initializer_list<T> ilist );
(2)

Replaces the contents of the container.

1) Replaces the contents with copies of those in the range [first, last). Only unique elements are inserted, ensuring no duplicates are stored in the container.
If either argument is an iterator into *this, the behavior is undefined.

This overload participates in overload resolution only if InputIt satisfies LegacyInputIterator.

2) Replaces the contents with the elements from ilist. Only unique elements are inserted, ensuring no duplicates are stored in the container.

All iterators (including the cend() iterator) and all references to the elements are invalidated.

Parameters

count - the new size of the container
value - the value to initialize elements of the container with
first, last - the range to copy the elements from
ilist - std::initializer_list to copy the values from

Complexity

1)
Average case: linear (i.e. O(N + M), where N is std::distance(first, last), where N is std::distance(first, last) and M is the elements in the original deque_of_unique.
Worst case: quadratic, i.e. O(N2).
2)
Average case O(N) (N is std::size(init)).
Worst case: O(N2).
