std:: deque_of_unique

Defined in header <deque_of_unique>
template <

   class  T,
   class  Hash  = std::hash
   class  KeyEqual  = std::equal_to
> class deque_of_unique ;

Combining the characteristics of a double-ended queue (deque) and a set, the deque_of_unique ensures that all elements are unique and can be accessed or modified from both ends. This makes it perfect for applications where maintaining order and preventing duplicates is essential.

Example

#include <deque>
#include <iostream>
#include "deque_of_unique.h"
 
int main()
{
    // Create a deque_of_unique containing integers
    containerofunique::deque_of_unique<int> und = {3, 5, 20, 9};
     
    // Add an integer to the beginning and end of the deque_of_unique
    und.push_front(3);
    und.push_back(28);
     
    // print values of deque_of_unique
    for (int n : und.deque())
    std::cout
    << n << ' ';
    std::cout
    << '\n';
}
