Constructs a new deque_of_unique from a variety of data sources.

1) The default constructor. Constructs an empty deque_of_unique.
2) The default constructor. Constructs an empty deque_of_unique.
3) Constructs a deque_of_unique with the contents of the range [first, last) and inserting only the unique elements from the given range into the container. Duplicate elements are automatically excluded, and only the first occurrence of each element is retained. Each iterator in [first, last) is dereferenced exactly once.
4) Equivalent to deque_of_unique(il.begin(), il.end()).
5) The copy constructor. Constructs a deque_of_unique with the contents of other.
6) The move constructor. Constructs a deque_of_unique with the contents of other.

Parameters

value - The value of the element to append.
first, last - the range to copy the elements from
other - another container to be used as source to initialize the elements of the container with
init - initializer list to initialize the elements of the container with
