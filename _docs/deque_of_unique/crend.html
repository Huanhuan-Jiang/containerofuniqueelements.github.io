containderofuniqueentries::deque_of_unique:: crend 

const_reverse_iterator crend() const noexcept;

Returns a reverse iterator to the first element of the reversed deque_of_unique. It corresponds to the last element of the non-reversed deque_of_unique.

If the deque_of_unique is empty, the returned iterator is equal to rend().

[] Parameters

(none)

Return value

Iterator to the first element.
