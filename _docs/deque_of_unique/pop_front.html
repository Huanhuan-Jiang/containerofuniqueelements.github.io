void pop_front();

Removes the first element of the container. If there are no elements in the container, the behavior is undefined.

Iterators and references to the erased element are invalidated. If the element is the last element in the container, the cend() iterator is also invalidated. Other references and iterators are not affected.

Parameters

(none)

Return value

(none)

Complexity

Constant.

Exceptions

Throws nothing.
