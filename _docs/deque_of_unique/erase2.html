(2) (since C++20)
1) Erases the element that compare equal to value from the container. Equivalent to 
auto it = std::remove(c.cbegin(), c.cend(), value);
    auto r = c.cend() - it;
    c.erase(it, c.cend());
    return r;
2) Erases the element that satisfy the predicate pred from the container. Equivalent to 
auto it = std::remove_if(c.cbegin(), c.cend(), pred);
    auto r = c.ccend() - it;
    c.erase(it, c.ccend());
    return r;
