void pop_back();

Removes the last element of the container.

Calling pop_back on an empty container results in undefined behavior.

Iterators and references to the last element are invalidated. The cend() iterator is also invalidated.

Parameters

(none)

Return value

(none)

Complexity

Constant.

Exceptions

Throws nothing.
